# dbc-ipher

A TUI for viewing CAN bus DBC files, built in Rust with [ratatui](https://github.com/ratatui/ratatui).

[![CI](https://github.com/jacobg826/dbc-ipher/actions/workflows/ci.yml/badge.svg)](https://github.com/jacobg826/dbc-ipher/actions/workflows/ci.yml)
[![Last Commit](https://img.shields.io/github/last-commit/jacobg826/dbc-ipher)](https://github.com/jacobg826/dbc-ipher/commits/main/)
[![License: MIT](https://img.shields.io/badge/license-MIT-blue.svg)](LICENSE)

## What it does

Browse the message/signal hierarchy of a `.dbc` file from the terminal. Navigate with vim-style keybindings and inspect message/signal details.

**Status:** actively being developed.
